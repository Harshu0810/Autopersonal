# OCEAN Personality Analysis Platform

AI-powered Big Five personality assessment using text analysis and surveys.

## Setup

1. Create Supabase project and run schema.sql
2. Get Hugging Face API token
3. Copy .env.local.example to .env.local and fill values
4. Install: `npm install`
5. Run: `npm run dev`
6. Deploy to Vercel

## Environment Variables

See .env.local.example for required variables.
